# delete_dota2_extra_fonts

Данный код был создан для ситуаций, когда в доте выходят массовые обновления.
После них, обычно, такие вещи как шрифты слетают, а лезть каждый раз в проводник, искать файл... Не всегда же хочется.
В таких ситуациях можете просто запустить этот код и он самостоятельно удалит все ненужные файлы.
В случаях, если ваш путь отличается, или вам нужно оставить другой шрифт, то просто меняете эти параметры прямо в коде.

Копируете проект в ваше ide:
```
git clone https://github.com/mailorq/delete_dota2_extra_fonts.git
```

Активируете venv
```
python -m venv .venv
.\.venv\Scripts\activate
```

Заходите в директорию файла:
```
cd delete_dota2_extra_fonts
```

Запуск
```
python app.py
```
